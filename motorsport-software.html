Motorsport Technical Management Software

Internal engineering software developed for motorsport technical data management and analysis.

FDC Diseño y Desarrollo | 2025 – Present

Overview

This project consists of a motorsport-oriented technical software platform developed to support the organization, storage, and analysis of engineering data in competition environments. The application was designed to centralize information related to vehicle set-up, tyre usage, aerodynamic configuration, track sessions, and circuit studies into a structured and accessible workflow.

The Challenge

In motorsport engineering, technical information is often scattered across spreadsheets, notes, setup sheets, and session records, making it difficult to maintain consistency, compare configurations, and quickly retrieve relevant data. The objective of this project was to build a dedicated software tool capable of improving traceability, workflow efficiency, and engineering access to critical competition data.

My Role

I was responsible for the development and engineering definition of the tool, including software structure, interface design, database organization, and the implementation of key technical modules aligned with competition engineering needs.

Main Features

  • Set-up configuration management
  • Tyre registration and usage history tracking
  • Aerodynamic parameter and configuration modules
  • Event and session organization
  • Sector-based lap analysis and engineering comparison tools
  • Database-backed storage and retrieval of technical information

Technical Approach

The application was developed in Python as a desktop engineering tool with a graphical user interface and integrated database structure. The architecture was designed to support modular growth, allowing the addition of new analysis panels and technical workflows as project requirements evolved.
